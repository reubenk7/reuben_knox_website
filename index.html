PORTFOLIO

Otago Mentor Match Project

A project I am currently working on in my final semester at University as part of a team of five using Agile and SCRUM methodologies to develop a web application for the Otago University Mentor Match program. The application will allow for Mentees and Mentors to sign up and for Mentees to then be able to filter down potential Mentors to request and also allow Admins to manage the system and generate reports. We are developing with PHP, HTML, CSS, and some Javascript with a MySQL database.

Māori Opportunities Project

A hypothetical project I developed as part of a Agile team of four to create a web application to help Māori students find research opportunities, attend events, and liaise with staff. We developed the web application using HTML, CSS, and Vue.js for the Frontend and Java with JDBI connecting to an H2 Database for the Backend

Product Management System and Shopping site project

A project I developed on my own for the INFO202 paper it has a desktop application for managing products (Create, View, Edit, Delete) and a web application shopping site for customers to purchase products. Both are connected to the same database so when a customer buys an item the quality in stock is automatically updated in the product management system. I developed this project using Java with Jdbi connecting to an H2 Database for the Backend with a Swing GUI for the product management interface, for the Frontend of the Shopping site I used HTML, CSS, and Vue.js

Fantasy Premier League Table Discord Bot

I personal project I made to create a Discord Bot which can get the results of a specified Fantasy Premier League mini league when called and reply with a Discord Embed of the results. I developed the Discord bot using the Node.js module Discord.js and store the saved leagues known to the bot in a JSON file so that the user doesn't have to get the leagues ID code everytime they want to call the bot. If I were to upscale the bot to allow for more users I could save the league IDs in a database instead.
